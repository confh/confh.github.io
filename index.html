Getting Started

DiscordApiWrapper is a high-performance Discord bot library designed for maximum speed and minimal overhead. It provides essential features for interacting with the Discord API.

Blazing Fast

Optimized for low-latency communication with Discord's API

Type Safe

Written in TypeScript with complete type definitions

Modern

Built with modern JavaScript practices and patterns

Lightweight

Keeps your bot's memory usage minimal

Installation

npm install @confis/discordapiwrapper
yarn add @confis/discordapiwrapper
npx jsr add @confis/discordapiwrapper

Client

The Client class is the main entry point for interacting with the Discord API. It handles the WebSocket connection, authentication, and event dispatching.

import { Client, Intents } from "@confis/discordapiwrapper";

const client = new Client("YOUR_BOT_TOKEN", {
    intents: [
        Intents.GUILDS,
        Intents.GUILD_MESSAGES,
        Intents.GUILD_MEMBERS
    ]
});

client.connect();

Events

The library provides a comprehensive event system that allows you to respond to various Discord events.

// Basic event handling
client.on("ready", () => {
    console.log(`Logged in as ${client.user.displayName}`);
});

// Message events
client.on("messageCreate", (message) => {
    if (message.content === "!ping") {
        message.reply("Pong!");
    }
});

// Member events
client.on("memberJoin", (member) => {
    console.log(`${member.user.username} joined the server!`);
});

Commands

Create and manage Slash Commands, Context Menus, and Message Components with ease.

// Create a slash command
const command = new SlashCommandBuilder()
    .setName("greet")
    .setDescription("Sends a greeting")
    .addUserOption(option =>
        option
            .setName("user")
            .setDescription("User to greet")
            .setRequired(true)
    );

// Register the command
client.setGlobalCommands([command]);

// Handle the command
client.on("interactionCreate", async (interaction) => {
    if (interaction instanceof SlashCommandInteraction) {
        if (interaction.name === "greet") {
            const user = interaction.options.getUser("user");
            await interaction.reply(`Hello, ${user.username}!`);
        }
    }
});

Message Components

Add interactive elements to your messages using buttons, select menus, and more.

// Create a button
const button = new ButtonBuilder()
    .setLabel("Click me!")
    .setStyle(ButtonStyles.PRIMARY)
    .setCustomId("my-button");

// Create an action row
const row = new ActionRowBuilder()
    .addComponents(button);

// Send message with components
channel.send({
    content: "Here's an interactive message!",
    components: [row]
});

// Handle button interactions
client.on("interactionCreate", (interaction) => {
    if (interaction instanceof ButtonInteraction) {
        if (interaction.customId === "my-button") {
            interaction.reply("Button clicked!");
        }
    }
});

Modals

Create interactive modal forms with text inputs for user interaction.

// Create a modal with text input
const modal = new ModalBuilder()
    .setTitle("User Information")
    .setCustomId("user-info-modal")
    .addComponents(
        new TextInputBuilder()
            .setCustomId("name")
            .setLabel("Name")
            .setStyle(TextInputStyles.SHORT)
            .setMinLength(1)
            .setMaxLength(4000)
            .setPlaceholder("John")
            .setRequired(true),
        new TextInputBuilder()
            .setCustomId("bio")
            .setLabel("Biography")
            .setStyle(TextInputStyles.PARAGRAPH)
            .setPlaceholder("Tell us about yourself...")
            .setRequired(false)
    );

// Show the modal in response to an interaction
interaction.showModal(modal);

// Handle modal submissions
client.on("interactionCreate", async (interaction) => {
    if (interaction instanceof ModalInteraction) {
        if (interaction.data.custom_id === "user-info-modal") {
            const name = interaction.data.components[0].data.value;
            const bio = interaction.data.components[1].data.value;
            
            await interaction.reply(`Name: ${name}\nBio: ${bio}`);
        }
    }
});

Embeds

Create rich embed messages with customizable fields, images, and formatting.

const embed = new EmbedBuilder()

channel.send({ embeds: [embed] });

Collectors

Collectors allow you to gather interactions or responses over a period of time. They're particularly useful for creating interactive menus, polls, or multi-step processes.

// Create a button
const button = new ButtonBuilder()
    .setLabel("Click me!")
    .setStyle(ButtonStyles.PRIMARY)
    .setCustomId("collect-button");

// Send message with button
const message = await channel.send({
    content: "This message will collect button clicks for 30 seconds!",
    components: [new ActionRowBuilder().addComponents(button)]
});

// Create a collector
const collector = message.createComponentCollector({
    timeout: 30000, // 30 seconds
    component_type: [ComponentTypes.BUTTON],
    filter: (i) => i.user.id === message.author.id // Only collect interactions from message author
});

// Listen for collected interactions
collector.on("collect", (type, interaction) => {
    console.log(`Collected a ${type} interaction!`);
    interaction.reply("Button click collected!");
});

// Listen for when the collector ends
collector.on("end", () => {
    console.log("Collector ended!");
    message.edit({
        content: "This collector has ended.",
        components: [] // Remove components
    });
});

// You can also reset the collector's timer
collector.resetTimer();

// Or end the collector early
collector.end();

Collector Options

  • timeout: Duration in milliseconds before the collector automatically ends
  • component_type: Type of component to collect (BUTTON, STRING_SELECT, etc.)
  • filter: Function to filter which interactions should be collected

Collector Events

  • collect: Emitted when an interaction is collected
  • end: Emitted when the collector ends (timeout or manual)
